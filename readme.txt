■気づきにくいかもしれない機能

	GUI編集時、カーソル描画以外のときでも、
	Shiftキーを押しながらドラッグするとマップ移動が出来ます。

	マウスをミドルクリックすると、
	そこからテスト実行できます。(そういうオプションがオンの時だけ、デフォルトはオン)

	プロパティエディタの複数行入力は
	Shift+Enterで行えます。

	グラフィカルデザイナでは、ポイントした地点の直下のチップが
	左下に表示されます。そのまま右クリックするとそのチップを描画チップにできます。

	Ctrl+Shiftでグラフィカルデザイナとチップリストのフォーカスを
	切り替えることができます。

	カーソルキーでグラフィカルデザイナを移動したり、クラシックチップリストの
	カーソルを移動できたりします。

	プロパティリストはダブルクリック以外にもF2で編集に入る事ができます。

	α値付きのpng画像もチップとして利用できます。

	グラフィカルデザイナはホイールを使って移動できます。
	Shiftキーを押しながらスクロールすると横方向へ移動します。

	オーバービューは、Spaceキー表示の時はモーダルダイアログですが、
	メニューから呼び出しの時はモードレスダイアログとなります。

	BGMや効果音の項目名をダブルクリックで効果音、BGMを再生します。同じ項目をダブルクリックすると再生終了します。

	カスタムパーツは一覧画面からパーツを右クリックでコピーや削除ができます。

	カスタムパーツの「チコリン はっぱカッター」の攻撃タイミングは項目名をダブルクリックで設定を削除できます。

■未実装だったり動きが悪かったり既知のバグだったりすること
　（XYZさんに書いていただいたものも載せています。ありがとうございました。）

	時々プロジェクトファイルがおかしくなり、テスト実行でエラーが出る
		→プロジェクトファイルを削除し、作り直してください。

	時々Sideを起動する際、エラーが出る
		→Sideフォルダの「side.xml」を削除してください。
		　（この際、一部の設定が失われます）

	テキストエディタを使用し、内部ブラウザでテスト実行した後に
	グラフィカルデザイナを使用すると少し前のデータに戻ってしまう
		→データを保存せずに、Sideを再起動してください。

	テキストエディタを使った後グラフィカルデザイナを使用するとテキストエディタでの編集前に戻ってしまう
		→面倒な方法ですが，テキストエディタを使用した場合はSideのバグ上，
		　グラフィカルデザイナ(以後GDI)を使用するとデータが初めてテキストエディタを開いたとき(?)
		　に戻ってしないますので
		　テキストエディタで編集→テスト実行は可能→GDI使いたいならGDI出す前に保存して再起動
		　この方法を使うとおそらく大丈夫です。

	パターン画像・マップチップ画像が編集できない
		→SideがGDI用にそれらの画像を占領しているからです｡
		　Sideを一旦終了するか、別のプロジェクトを開いてください。

	現在のプロジェクトのランタイムを再設定(R)...が全くもって使えない
		→Side自身がpattern.gif, mapchip.gif, ex.pngあたりを使用していて
		　Sideがそれを上書きしようとしているのにSideが拒否しているからです。

	GDI編集時，拡大率を200%にすると動作が重くなる
		→拡大率を200%にすると一部の環境では動作が重くなるので使用するのを控えることをお勧めします。

	プロパティ項目のマップチップやパターン画像を指定するところで
	別のフォルダの画像でマップチップやパターン画像を上書きしようとすると
	「(×)ファイルをコピー出来ませんでした。ファイルのコピー先を再指定してください｡」とでて，
	元の画像を指定すると「上書きはできません。元のファイルを消すか，別のファイルを指定してください。」
	と出てくる。
		→これはランタイム再設定のときと同じことが発生しています｡
		　別の名前を指定すればいいのですが，どうしても元のファイルを上書きしたい場合は，
		　一度Sideを終了し，手動で上書きしてください。

	完成したあとにHTMLをブラウザで開くと初めに変なところに正男がいたり
	ステージ1以外からスタートしてしまう
		→これはSideの不具合ではありません。テスト実行時のHTMLを直接ブラウザで開いているのが原因です。
		　Sideから，「HTML書き出し」を行ってください。

	外部ブラウザでテストプレイができない
		→Canvas正男は最新のブラウザでプレイ可能です。
		　Internet Explorer9より前のブラウザをお使いの方はブラウザをアップデートしてください。（Internet Explorerは2022年6月にサポート終了しました。）

	GIFアニメ、MIDI、AUファイルが使用できない
		→今のところCanvas正男の仕様です。

	クラシックチップリストのボスのチップに縦線・横線のゴミが出る。
		→エディタの設定で「クラシックチップリストのチップを補完描画する」をオフにして下さい。

	OGGファイルの効果音、BGMが再生できない。
		→内部でWindows Media Playerを利用して再生しているのでLAVFilters（https://github.com/Nevcairiel/LAVFilters/releases）をインストールすると再生できるようになります。
